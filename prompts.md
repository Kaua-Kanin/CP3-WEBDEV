# Prompts utilizados – CP3 CRUD e IA Generativa

## IA 1 – ChatGPT

### Prompt utilizado

Crie uma aplicação web usando apenas HTML, CSS e JavaScript puro (sem frameworks ou bibliotecas externas).

Tema: lista de filmes e séries.

Requisitos obrigatórios:

- Tela de login inicial
- Usuário: aluno
- Senha: fiap2025
- Validar campos vazios
- Exibir mensagem de erro na tela
- Após login correto, esconder login e mostrar CRUD

CRUD:
- Usar um array de strings
- Iniciar com pelo menos 3 itens
- Adicionar item no final da lista
- Adicionar item no início da lista
- Listar itens dinamicamente na tela
- Editar item individualmente
- Remover item individualmente usando índice

Validações:
- Não permitir item vazio
- Se cancelar edição ou deixar vazio, manter item original
- Toda alteração deve atualizar a tela automaticamente

Organize toda a lógica em funções nomeadas.

Não use frameworks nem recursos avançados além de HTML, CSS e JavaScript básico.

### Pontos positivos

* Código simples e organizado
* HTML, CSS e JavaScript separados
* CRUD completo
* Fácil entendimento e explicação
* Próximo dos conteúdos vistos em aula

### Problemas encontrados

* Necessidade de pequenos ajustes visuais no CSS
* Ajustes simples nas validações

## IA escolhida como base

Escolhi o ChatGPT como base porque apresentou a solução mais próxima dos requisitos da atividade e dos conteúdos vistos em aula e por ser a IA que uso frequentemente. O código ficou mais simples de entender, explicar e adaptar, além de estar organizado em arquivos separados (HTML, CSS e JavaScript). Após isso, realizei pequenos ajustes no visual e nas validações para atender completamente aos critérios da CP.
